规则执行优先级，数字越大优先级越高，范围 1-10，建议：高风险规则设为 8-10，常规规则设为 5-7

规则适用的边界，global 表示全局适用，也可以设置为特定场景如 marketing_customer_service_assistant

📊 认知参数说明

          

            
遗忘曲线：控制知识随时间衰减的速度。值越大，知识忘得越快。

            
严谨度边界：控制知识边界匹配的严格程度。值越大，越严格。

            
疑心病指数：控制冲突检测的敏感度。值越大，越容易检测到冲突。

            
最低置信度：过滤低质量知识。值越大，保留的知识越可靠。
