# 9 个真实场景执行矩阵（个人知识服务）

统一逻辑：**身份 → 核心工作 → 痛点 → 工具优化路径 → 价值指标 → 功能清单 → 验证标准**

## 1) 慢病患者管理师（health_chronic_care_manager）
- 核心工作：阅读体检报告/用药说明，生成每日管理要点。
- 痛点：报告条目分散，复诊前复盘慢。
- 工具路径：PDF上传→条款分块→规则推理。
- 价值指标：检索效率 +60%，复盘时间 -45%。
- 功能清单：文档上传、知识分块、风险提示规则。
- 验证标准：样本>=1600，决策命中率>90%。

## 2) 家庭营养管理师（health_nutrition_planner）
- 核心工作：根据营养指南和家庭健康记录排周计划。
- 痛点：多来源标准冲突、人工汇总慢。
- 优化：DOC/PDF入库 + 冲突检测。
- 指标：方案生成速度 +55%，冲突遗漏率 -40%。
- 清单：CONTRADICTION 检测、批量导入。
- 标准：样本>=1500，冲突可追溯率=100%。

## 3) 专注力教练（efficiency_focus_coach）
- 核心工作：从效率方法文档中制定执行清单。
- 痛点：方法建议泛化不足。
- 优化：文档证据驱动 + cognition loop。
- 指标：首响时间 -50%，建议可执行性提升。

## 4) 个人理财顾问（finance_wealth_advisor）
- 核心工作：对照监管与产品说明形成建议。
- 痛点：条款查阅费时，口径不一致。
- 优化：制度文件上传、规则约束输出。
- 指标：检索效率 +65%，错误率 -35%。

## 5) 保险咨询师（finance_insurance_consultant）
- 核心工作：保单条款比对与问答。
- 痛点：条款版本多、响应慢。
- 优化：文档结构化 + loop runs 复核。
- 指标：响应时长 -55%。

## 6) 考试备考教练（education_exam_coach）
- 核心工作：教材大纲解析，形成复习计划。
- 痛点：章节重难点提取低效。
- 优化：DOC/PDF解析 + chunk知识映射。
- 指标：计划输出时间 -55%。

## 7) 职业转型顾问（career_transition_advisor）
- 核心工作：岗位JD/能力框架映射。
- 痛点：能力差距识别慢。
- 优化：多文档入库 + 规则优先级推理。
- 指标：路径建议稳定性 +30%。

## 8) 项目复盘教练（career_project_retrospective）
- 核心工作：会议纪要与SOP复盘。
- 痛点：结论复现困难。
- 优化：Judgements + Loop Runs 审计。
- 指标：复盘一致性 +40%。

## 9) 营销客服助手（marketing_customer_service_assistant）
- 核心工作：查阅国家/企业/部门规章并回复客户。
- 痛点：查阅慢、记忆偏差、响应不及时。
- 优化：DOC/PDF规章上传→结构化知识→strict_policy_response_mode。
- 指标：检索耗时 18.0min→4.7min；首响 140s→46s；错误率 19%→6%。

---

## 三轮迭代闭环执行标准
1. 复盘偏差：记录问题与缺口。
2. 优化落地：功能改造与接口上线。
3. 验证效果：大样本实验与报告落盘。

所有场景遵循：样本规模 >1000，输出可复现，指标可对比，结果可审计。
